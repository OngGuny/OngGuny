### Architecture Experience

| 영역 | 설계 경험 |
|------|----------|
| **RAG Pipeline** | 문서 추출 → 청킹 → 임베딩 → pgvector 벡터 검색 → LLM 응답 생성 |
| **Airflow Crawler** | 이종 소스 통합 수집, Fetcher 전략 패턴, 배치 스케줄링 자동화 |
| **Vector Dedup** | 2단계 임베딩 기반 의미론적 중복 판별 (제목 유사도 → 본문 유사도 + 길이 비율) |
| **NLI Scoring** | 다축 품질 스코어링 — NLI 관련도, 최신성, 신뢰도, 중복 감점 |
| **Agentic Cost Optimization** | 태스크 복잡도 기반 LLM 경량/고성능 모델 동적 분배로 운영 비용 절감 |
| **Graceful Degradation** | 외부 서비스 장애 시 상태 코드 분리 (재시도 가능 vs 영구 실패) |
